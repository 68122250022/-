1. โปรแกรมตรวจสอบเลขคู่/เลขคี่
  ชื่อโปรแกรม: EvenOddCheck.java

  Input: รับค่าตัวเลขจำนวนเต็ม (Integer) จากผู้ใช้ 1 จำนวน

  Process: นำตัวเลขที่รับมาหารเอาเศษด้วย 2 (number % 2) หากเศษเท่ากับ 0 แสดงว่าเป็นเลขคู่ หากไม่ใช่แสดงว่าเป็นเลขคี่

  Output: แสดงข้อความบอกว่าตัวเลขนั้นเป็นเลขคู่ (Even) หรือ เลขคี่ (Odd)


2. โปรแกรมคำนวณคะแนนรวมและผ่าน/ไม่ผ่าน
  ชื่อโปรแกรม: PassFailCheck.java

  Input: รับค่าคะแนนของนักศึกษา

  Process: ตรวจสอบเงื่อนไขด้วยคำสั่ง if-else โดยนำคะแนนไปเทียบกับเกณฑ์ที่กำหนด (เช่น ถ้าคะแนน >= 50 ถือว่าผ่าน)

  Output: แสดงคะแนนที่กรอกเข้ามา และข้อความแจ้งผลว่า "ผ่าน (Pass)" หรือ "ไม่ผ่าน (Fail)"


3. โปรแกรมเกี่ยวกับ Array (หาผลรวมและค่าเฉลี่ย)
  ชื่อโปรแกรม: ArrayScore.java

  Input: รับค่าคะแนนของนักศึกษาจำนวน 5 คน (ทีละคน) เก็บลงใน Array

  Process: ใช้ลูป for วนรับค่าและนำคะแนนใน Array มาบวกสะสมไว้ในตัวแปร sum จากนั้นคำนวณค่าเฉลี่ยโดยนำผลรวมหารด้วย 5 (sum / 5.0)

  Output: แสดงผลคะแนนรวม (Total score) และคะแนนเฉลี่ย (Average score)


4. โปรแกรม Mini Challenge (นับจำนวนเลขคู่และเลขคี่)
  ชื่อโปรแกรม: MiniChallenge.java

  Input: รับค่าตัวเลขจำนวนเต็ม 10 จำนวน เก็บไว้ใน Array

  Process: ใช้ลูป for ตรวจสอบข้อมูลใน Array ทีละตัวด้วยเงื่อนไข if (numbers[i] % 2 == 0) หากเป็นจริงให้เพิ่มค่าตัวนับเลขคู่ (evenCount) หากเป็นเท็จให้เพิ่มค่าตัวนับเลขคี่ (oddCount)

  Output: แสดงจำนวนรวมของเลขคู่ (Even count) และจำนวนรวมของเลขคี่ (Odd count)


วิธีรันโปรแกรม
เปิด Command Prompt (Windows) หรือ Terminal (Mac/Linux)
ใช้คำสั่ง cd เพื่อเปลี่ยนไปยังโฟลเดอร์ที่เก็บไฟล์ .java ไว้
พิมพ์คำสั่งคอมไพล์โปรแกรม: javac ชื่อไฟล์.java (เช่น javac MiniChallenge.java)
พิมพ์คำสั่งรันโปรแกรม: java ชื่อคลาส (เช่น java MiniChallenge)


สิ่งที่ได้เรียนรู้จากการทำใบงานนี้
เข้าใจโครงสร้างพื้นฐานและการรับค่า/แสดงผลในภาษา Java
สามารถใช้เงื่อนไข if-else ในการตัดสินใจและควบคุมทิศทางของโปรแกรมได้
เข้าในการใช้งาน Loop ร่วมกับ Array เพื่อจัดการกับข้อมูลที่มีจำนวนหลายตัวได้อย่างมีประสิทธิภาพ
ได้ฝึกฝนกระบวนการคิดเชิงตรรกะ (Logical Thinking) และการหาข้อผิดพลาดของโปรแกรม (Debugging)
